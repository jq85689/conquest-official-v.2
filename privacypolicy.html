4. Local Storage and App Data

The app stores small amounts of state locally in your browser using localStorage. This includes:

  • Application state (progress, level, streaks, completed tasks) to provide a persistent single-user experience on this device.
  • Cookie consent choice (conquest_cookie_consent) to remember whether you accepted or declined analytics.
  • A local admin-only private visits counter and settings (if you enable and use the admin panel). This data is stored only in your browser and is not transmitted to any server by default.

5. Admin Panel and Private Visits

The app includes an optional browser-local admin panel protected by a PIN that allows the site owner to view and manage a private visits counter. The admin panel and its data are stored only in your browser's localStorage and are not shared unless you export them manually. If you enable "Auto-track" in the admin settings, the private counter will increment only for users who accepted analytics (cookie consent).
